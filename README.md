# 程式碼評測結果展示網頁

展示程式碼評測結果供查詢。

## 環境需求

Node.js v18.20.4 或以上版本

## 開發環境

### 1. clone 專案到本地

```sh
git clone https://github.com/nuk-csie-114-Student-and-AI-Coding/nuk-csie-114-Student-and-AI-Coding.github.io.git
```

### 2. 安裝專案套件

**切換到專案目錄下**，執行以下指令

```sh
npm install
```

### 3.開發模式 (即時更新)

用於開發或測試功能，執行以下指令

```sh
npm run dev
```

## 部署到 GitHub Pages

將專案 **建置並部署** 到 GitHub Pages，執行以下指令

```sh
npm run deploy
```
